# Workflow Analyse statique de code

Ce workflow organise une campagne d'analyse statique en cinq phases :

1. cadrage et politique d'analyse ;
2. préparation technique ;
3. exécution des analyses ;
4. qualification des résultats ;
5. remédiation et suivi.

Validation :

```powershell
py scripts/validate_workflow_data.py `
  data/workflows/analyse-statique-code/manifest.json
```
